## Quick wins: Write tests

1. Pick a [test issue](https://github.com/thechibuikem/kronos/issues?q=label%3Atests)
2. Fork the repo
3. Write tests following the [test contribution template](.github/ISSUE_TEMPLATE/test_contribution.md)
4. Run `npm run test` to verify
5. Open a PR with title `Tests: [title: "Tests: [Module/Controller Name]"]` and link the issue
6. We'll review and merge

See [auth tests](backend/modules/auth/tests) for examples.
